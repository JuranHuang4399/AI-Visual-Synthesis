# GIF资源目录

请将你的像素GIF动画文件放在这个目录下。

## 建议的文件命名
- `pixel-char-1.gif` - 角色动画1
- `pixel-char-2.gif` - 角色动画2
- `pixel-char-3.gif` - 角色动画3
- `pixel-decor-1.gif` - 装饰动画1
- `pixel-decor-2.gif` - 装饰动画2
- `pixel-star.gif` - 星星动画
- `pixel-heart.gif` - 爱心动画

## 使用建议
- 文件大小：建议每个GIF < 500KB
- 尺寸：装饰用建议 64x64 或 128x128
- 循环：建议设置为无限循环
- 背景：建议透明背景（PNG格式的GIF）
